# Social Home (stable)

Stable channel of the Social Home Home Assistant add-on. Tracks
[CalVer releases](https://github.com/social-home-io/ha-app/releases)
of the upstream `social-home-io/socialhome` Docker image.

See [`DOCS.md`](DOCS.md) for the in-app documentation rendered by
HA, and [the repository
README](https://github.com/social-home-io/ha-app#readme) for
install instructions.
